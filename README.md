# Gruvbox Material for VS Code


## Description
Fork of the Gruvbox Material theme by [sainnhe](https://github.com/sainnhe/gruvbox-material-vscode)

## Features

- Notebook support.
- Foreground colors with higher contrast.
- Highlighting consistent with the vim [version](https://github.com/sainnhe/gruvbox-material)
- Semantic highlighting support.
- Italic support.
